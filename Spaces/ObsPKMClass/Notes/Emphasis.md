#### Emphasis

Surround the text with the following symbols, eg. `_italic_` produces _italic_.

| Effect        | Symbol   | Display     |
| ------------- | -------- | ----------- |
| Italics       | _ or *   | _Example_   |
| Bold          | __ or ** | __Example__ |
| Highlight     | ==       | ==Example== |
| Strikethrouth | ~~       | ~~Example~~            |
